Pendahuluan

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML memberikan struktur pada halaman web dengan menggunakan elemen-elemen seperti paragraf, heading, tautan, dan gambar. Tutorial ini akan memandu Anda melalui dasar-dasar coding HTML.

Struktur Dasar HTML

Setiap dokumen HTML dimulai dengan struktur dasar berikut:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang di Halaman Web Saya</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Penjelasan Struktur

  • <!DOCTYPE html>: Menunjukkan bahwa ini adalah dokumen HTML5.
  • <html>: Elemen root dari halaman HTML.
  • <head>: Berisi informasi metadata tentang halaman, seperti <title> dan link ke stylesheet.
  • <title>: Menetapkan judul halaman yang ditampilkan di tab browser.
  • <body>: Berisi konten utama halaman yang akan ditampilkan di browser.

Elemen HTML Umum

Heading

Elemen heading digunakan untuk membuat judul. Ada enam tingkat heading dari <h1> (paling penting) hingga <h6> (paling tidak penting):

<h1>Judul Utama</h1>
<h2>Subjudul</h2>
<h3>Sub-subjudul</h3>

Paragraf

Gunakan elemen <p> untuk membuat paragraf:

<p>Ini adalah paragraf pertama saya.</p>
<p>Ini adalah paragraf kedua saya.</p>

Tautan

Gunakan elemen <a> atau yang lainnya untuk membuat tautan. Gunakan atribut href untuk menentukan URL tujuan:

<a href="https://www.example.com">Kunjungi Example.com</a>

Gambar

Untuk menambahkan gambar, gunakan elemen <img> dengan atribut src yang menunjuk ke file gambar dan alt untuk teks alternatif:

<img src="gambar.jpg" alt="Deskripsi Gambar">

Daftar

Ada dua jenis daftar: daftar berurutan (<ol>) dan daftar tidak berurutan (<ul>). Daftar item ditandai dengan <li>:

Daftar Tidak Berurutan:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Daftar Berurutan:

<ol>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
</ol>

Formulir

Formulir memungkinkan pengguna untuk mengirimkan data. Gunakan elemen <form>, <input>, <label>, dan <button> untuk membuat formulir sederhana:

<form action="/submit" method="post">
    <label for="name">Nama:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Kirim</button>
</form>

Tips dan Trik

  • Validasi HTML: Gunakan alat validasi HTML seperti W3C Validator untuk memastikan kode Anda bebas dari kesalahan.
  • Keterbacaan: Gunakan indentasi dan pemformatan yang konsisten untuk meningkatkan keterbacaan kode Anda.
  • Komentar: Tambahkan komentar menggunakan <!-- komentar --> untuk menjelaskan bagian-bagian penting dalam kode Anda.
